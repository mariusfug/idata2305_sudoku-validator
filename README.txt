Sudoku validator Gruppe 13


Compiler:
 - Vi har brukt GCC compiler og GDB debugger med MinGW i dette prosjektet, 
   ref: https://code.visualstudio.com/docs/cpp/config-mingw (Til og med kap. "Running the build")


Build sudoku.c i VSC:
 - Terminal -> Configure Default Build Task -> g++.exe build active file
   tasks.json burde bli opprettet


Kjør prosjektet:
 - Terminal -> Run Build Task
   sudoku.exe burde bli opprettet i filmappen

 - Naviger til prosjektmappen i terminal

 - Fra prosjektmappen, kjør programmet med kommandoen .\sudoku.exe <filnavn>.csv
